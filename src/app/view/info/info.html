Este aplicativo tem a finalidade de auxiliar no seu estudo da Lingua Inglesa, apresentando uma proposta de estudo de 100 dias.
Nesses 100 dias você terá a oportunidade de conhecer mais de 3000 palavras da Lingua Inglesa.
Site hospedado no Github.
Site desenvolvido utilizando o Frameword Angular.
Utilizamos HTML, Javascript e CSS para o desenvolvimento do Site.
